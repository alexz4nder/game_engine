# game_engine
Game engine i work on in my free time . It uses vulkan api for graphics.
#### Some features i implemented or plan to implement:
- [x] clustered forward rendering 
- [x] normal mapping
- [ ] ambient occlusion
- [ ] animations
- [ ] particle system
- [x] loading gltf scene
- [ ] exporting gltf scene
- [ ] scene editor (work in progress)
- [ ] scripting system

#### dependencies:
- fastgltf
- vk-bootstrap
- VulkanMemoryAllocator-hpp
- flecs
- imgui
- stb image
- SDL3
- fmt
